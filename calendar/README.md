# 关于相遇与等待的纪念册

## 项目介绍
这是一个专为记录情侣设计的日历网页应用，承载着一段关于相遇与等待的故事。当两个灵魂在时光里交汇，那些三观的契合、深夜长谈的坦诚、彼此照亮的瞬间，都值得被温柔记录。应用旨在帮助记录和管理两人之间的重要日期、回忆瞬间、倒计时事件以及爱情箴言，让这份特殊的情感轨迹以更清晰的方式存在。

## 核心功能
- **重要日期记录**：可保存首次约会时间、确定关系时间等关键时间点，让每一个重要节点都有迹可循
- **回忆收藏**：记录相处中的美好回忆，并按时间分类展示，留存那些鲜活被理解的瞬间
- **爱情箴言**：存储彼此喜欢的情话或感悟，让温暖的话语成为时光里的锚点
- **倒计时管理**：为重要纪念日、下次见面等设置倒计时提醒，让等待的日子也充满期待
- **时间筛选**：可按年月筛选查看特定时间段的回忆，在时光碎片中重温过往

## 数据存储
应用使用`localStorage`存储所有数据，包括：
- 约会时间（meetTime）
- 恋爱时间（loveTime）
- 情侣姓名（coupleNames）
- 回忆列表（memories）
- 名言列表（quotes）
- 倒计时事件（countdowns）

所有数据保存在浏览器本地，确保隐私安全的同时，也能在同一设备上持久化保存这份独特的情感印记。

## 使用方法
1. 打开网页即可自动加载已保存的数据，继续记录或回顾过往
2. 可添加、编辑或删除回忆、名言和倒计时事件，让记录保持鲜活
3. 通过月份筛选器查看不同时间段的回忆，在时光脉络中寻找共鸣
4. 关键时间点会自动计算并展示相关纪念日信息，让每一份等待都有具体的刻度

## 核心意义
这里记录的，是关于"相遇的意义"的思考——即使注定要经历等待，那些共享过的深夜、交换过的秘密、彼此照亮过的瞬间，本身就是存在的证明。正如曾在某个回答里写下的：**"相遇的意义，在于让你在某个平凡的午后突然想起时，会觉得原来自己曾那样鲜活地被理解过。"**

等待或许是漫长的，但有些值得的人，让这份等待有了重量。就像那些为了爱情做过的"疯狂"事——不是冲动的冒险，而是明知前路未知，仍愿意保持真诚的坚持。在这里，每一次记录都是一种温柔的守候：不急于索取结果，只愿在时光里慢慢陪伴，让故事自然生长。

## 致时光里的我们
这个空间会继续记录下去，不为强求一个结果，只为留存那些珍贵的共鸣。无论是最终水到渠成，还是成为彼此生命里特殊的印记，这些关于理解、坦诚与等待的瞬间，都已构成了相遇本身的意义。正如确信的那样：**"遇见你不容易，错过了会很可惜，因为一辈子太短，下辈子不一定能再遇见。"**

## 技术栈
- HTML5 结构搭建
- JavaScript 交互逻辑实现
- localStorage 本地数据存储
- 原生DOM操作与事件处理

## 部署说明
直接将`index.html`文件部署到任何静态网页服务器或托管平台（如GitHub Pages）即可使用，无需后端服务支持，让这份记录可以随时陪伴左右。
